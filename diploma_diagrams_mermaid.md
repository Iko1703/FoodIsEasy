# Диаграммы для §1.3 (Mermaid → draw.io / export PNG)

## Рисунок 1.3 — Жизненный цикл пользователя

```mermaid
flowchart LR
  A[Регистрация и профиль] --> B[Каталог и история]
  B --> C[Рекомендации]
  C --> D[План питания]
  D --> E[Корзина покупок]
  E --> F[Аналитика]
  F -.-> B
```

## Рисунок 1.4 — Персональная рекомендация

```mermaid
flowchart TD
  S[Блюдо из каталога] --> A{Аллерген в составе?}
  A -->|Да| X[Исключить: балл 0]
  A -->|Нет| B[Начисление балла]
  B --> C[Сортировка по убыванию]
  C --> D[Топ-N + причины]
```

## Рисунок 1.5 — Групповой подбор

```mermaid
flowchart TD
  G[Ограничения всех участников] --> F{Безопасно для всех?}
  F -->|Нет| R[Не рекомендуется]
  F -->|Да| P[Личный балл каждого]
  P --> M[Среднее по группе]
  M --> L[Выдача]
  M --> V[Голосование среди допустимых]
```

## Рисунок 1.6 — Потоки между модулями

```mermaid
flowchart LR
  PR[Профиль] --> REC[Рекомендации]
  CAT[Каталог] --> REC
  REC --> PLAN[План питания]
  PLAN --> CART[Корзина]
  HIST[История] --> REC
  HIST --> AN[Аналитика]
  GR[Группа] --> REC
  GR --> PLAN
```
