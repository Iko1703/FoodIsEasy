2.3.2 Ключевые таблицы и связи

users — учётные записи; delishies — каталог; delishies_products — состав блюда (M:N с products); user_cuisine_preferences, user_product_preferences — предпочтения и аллергии; meal_plans, meal_plan_entries — план (несколько записей на дату+meal_type); meal_history — факты питания; groups, group_members — группы; group_votes, group_vote_options, group_vote_ballots — голосования; shopping_lists, shopping_list_items — корзина по плану; feedbacks, favorite_delishies — отзывы и избранное.

Пример SQL для скрина данных:
SELECT d.title, p.name, dp.quantity_grams
FROM delishies_products dp
JOIN delishies d ON d.id = dp.delishies_id
JOIN products p ON p.id = dp.product_id
WHERE d.title = 'Борщ классический';

2.6 Тестирование

Автотесты — JUnit 5 + Spring Boot Test + Testcontainers (PostgreSQL 16). Каждый интеграционный тест поднимает изолированную БД, Flyway применяет миграции, затем проверяется сервисный слой.

Тестовые классы:
— MealPlanServiceTest — создание плана, несколько блюд в слоте, generate;
— RecommendationServiceTest — непустые рекомендации, score > 0;
— ProfileServiceTest — сохранение предпочтений;
— DelishiesRepositoryTest — выборка блюд с продуктами;
— FoodIsEasyApplicationTests — загрузка контекста.

Запуск (нужен запущенный Docker Desktop для Testcontainers):

cd C:\Users\Iko\Documents\FoodProject\FoodIsEasy
mvn test

Альтернатива — только один класс:
mvn test -Dtest=MealPlanServiceTest
